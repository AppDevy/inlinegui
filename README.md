- [DocPad](http://docpad.org) via the [inlinegui docpad plugin](http://docpad.org/p/inlinegui)

## Development

```
git clone https://github.com/webwrite/inlinegui.git inlinegui
cd inlinegui
npm install; npm install -g coffee-script docpad
docpad run
```
